# Components

This directory contains all reusable React components.

## Organization

- `common/` - Shared UI components (Header, Footer, Button, etc.)
- `features/` - Feature-specific components
- `forms/` - Form components and input fields
- `layout/` - Layout wrapper components

## Guidelines

- One component per file
- Export as default
- Use TypeScript with proper typing
- Include JSDoc comments for complex components
- Keep components focused and single-responsibility
- Co-locate styles with component

## Naming

- Use PascalCase: `UserCard.tsx`, `LoginForm.tsx`
- Avoid generic names like `Component.tsx`
